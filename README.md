# Lofi-desktop

This is an application to listen lofi music from [Lofi Girl](https://youtube.com/c/LofiGirl) on a pc

The application is built with [Electron](https://www.electronjs.org/)

## Installation

If you want to install the application, you can go to the [release section](https://github.com/Greensky-gs/lofi-desktop/releases) and get the latest version

You will find the `setup.exe` in the assets of the release
